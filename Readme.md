Descrição:

Página web informativa desenvolvida com foco em organização de layout, semântica HTML e estruturação de estilos em CSS. O projeto foi pensado inicialmente para visualização em desktop, priorizando clareza visual e código bem estruturado.

Objetivos do Projeto:

Aplicar boas práticas de HTML semântico

Organizar o CSS de forma clara e escalável

Criar um layout limpo e de fácil manutenção

Aprendizados:

Uso correto de tags semânticas como header, main e section

Separação adequada entre estrutura e estilização

Importância da organização do código para futuras manutenções

Tecnologias Utilizadas:

HTML5

CSS3
